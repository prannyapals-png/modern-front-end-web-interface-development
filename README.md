# Crown Health Clinic

Crown Health Clinic is a responsive front-end website prototype developed for a modern health service. This project was created using HTML5, CSS3, and vanilla JavaScript to demonstrate professional front-end development practices without using frameworks.

## Industry Chosen
Health

## Project Features
- Three interconnected pages: Home, Services, and Contact
- Semantic HTML5 structure
- Responsive mobile-first design
- Flexbox and CSS Grid layouts
- Interactive mobile navigation menu
- Service filter with dynamic content updates
- FAQ accordion
- Contact form validation with feedback
- Accessibility considerations such as labels, alt text, keyboard focus states, heading hierarchy, and readable colour contrast

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript

## Suggested GitHub Pages Deployment
1. Create a public GitHub repository
2. Upload all project files
3. Go to repository Settings
4. Select Pages
5. Choose the main branch and root folder
6. Save and copy the deployment link

## Presentation Talking Points
- Design decisions: calm health brand, responsive layout, simple patient journey
- Accessibility: semantic tags, labels, alt text, colour contrast, keyboard-friendly components
- Challenges and solutions: mobile navigation, responsive layout, JavaScript interactivity without frameworks
- Industry relevance: usable front-end prototype before backend investment
